# AI-Powered Business Intelligence Analyst

Ask questions about your business data in plain English and get instant insights, SQL queries, and visualizations using AI.

---

# Project Overview

This project is a Streamlit-based AI application that allows users to:

- Upload CSV business datasets to a MySQL database.
- Ask natural language questions about the data.
- Automatically generate SQL queries.
- View query results in interactive tables.
- Visualize data with automatically generated charts.

The application uses OpenAIâ€™s GPT API to translate user questions into SQL queries, making data analytics accessible even for non-technical users.

---

# Key Features

- CSV Upload: Users can upload CSV files directly through the sidebar.  
- Natural Language Queries: Ask questions like "Total sales by region" or "Monthly sales trend".  
- Automatic SQL Generation: Converts plain English questions into safe SQL queries.  
- Interactive Data Tables: Displays query results in a clean, scrollable table.  
- Charts & Visualizations: Automatically generates bar charts for 2-column query results.
